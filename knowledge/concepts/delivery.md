# Логістика та Доставка

Інтеграція з поштовими операторами України для забезпечення вибору відділень та розрахунку адрес.

## Нова Пошта
- **Компонент**: `src/components/NovaPoshtaSelector.jsx`.
- **API**: Прямі запити до JSON API НП.
- **Дані**: Вибір міста та відділення. Зберігається повна адреса з областю та районом.

## Укрпошта
- **Компонент**: `src/components/UkrPoshtaSelector.jsx`.
- **Proxy**: `src/app/api/ukrposhta/route.js`. 
    - Вирішує проблему CORS та приховує `Bearer` токен на бекенді.
    - Використовує `classifier` API для пошуку міст та індексів.
- **Дані**: Пошук населеного пункту за назвою та вибір індексу.

## Формат зберігання
Всі дані про доставку конкатенуються у рядок і записуються в поле `address` таблиці `orders`. Див. [[concepts/database-schema]].

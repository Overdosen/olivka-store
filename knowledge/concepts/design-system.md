# Дизайн-система Store Olivka

Проект заснований на принципах мінімалізму, естетики та чистоти, що відповідає ніші преміального одягу для немовлят.

## Бренд-кольори
- **Основний текст**: `#524f25` (Глибокий оливковий).
- **Фон**: `#faf9f6` (Кремовий).
- **Акцент**: `#c4a882` (Золото).

## Типографіка
- **Майстер-шрифт (Десктоп)**: `Century Gothic` (застосовується для заголовків та основного тексту).
- **Адаптивний шрифт (Мобільні)**: `Inter` (автоматично вмикається для екранів < 768px).

## Ключові принципи UI
- М'які заокруглення.
- Використання `framer-motion` для плавних переходів.
- Сувора ієрархія відступів (Tailwind-класи).

# Панель адміністратора

Адміністративний інтерфейс для керування замовленнями, клієнтами та контентом магазину.

## Доступ
- Маршрут: `/admin/`.
- Доступ обмежений через Supabase RLS та роль `is_admin` у системних профілях.

## Можливості
- **Замовлення**: Перегляд повної адреси доставки (див. [[concepts/delivery]]), зміна статусів, контроль складу товарів.
- **Клієнти**: База даних покупців з історією замовлень.
- **Товари**: Редагування асортименту та налаштувань категорій.

## Технічні деталі
- Код знаходиться в `src/app/admin/`.
- Використовує окремий UI-фреймворк (більш функціональний дизайн) через `src/admin.css`.

## Тестові та AI акаунти
Для автоматизованих перевірок та технічної підтримки використовується наступний акаунт:

# Редизайн футера та мікровізуальне вирівнювання

Системне оновлення футера магазину Olivka Store з фокусом на професійну подачу бренду та платіжних систем.

## Ключові принципи

- **Чистий дизайн:** Відмова від рамок та hover-ефектів для логотипів. Вони мають виглядати як інтегрована частина фону.
- **Оптична компенсація:** Використання `transform: translateY()` для вирівнювання логотипів з різними внутрішніми відступами (наприклад, Укрпошта).
- **Колірна автентичність:** Вимкнення CSS-блендингу для брендів з критичними кольорами (червоний NP, жовтий Укрпошти) при збереженні `mix-blend-mode: screen` для білих платіжних іконок.

## Технічні деталі

### Стек вирівнювання іконок
```css
.footer-all-logos {
  display: flex;
  align-items: flex-end; /* Вирівнювання по базовій лінії */
  gap: 0.5rem;
}

.footer-logo-badge--bottom {
  /* Незалежне зміщення без впливу на сусідів */
  transform: translateY(3px); 
}
```

### Проблема прозорості SVG
Для іконок з білим фоном на темному футері використовується:
```css
.footer-svg-logo {
  mix-blend-mode: screen;
}
```
Для кольорових логотипів (Нова Пошта, Укрпошта, 24pay) використовується клас-виключення:
```css
.footer-svg-logo--no-blend {
  mix-blend-mode: normal;
}
```

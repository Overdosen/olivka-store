# Загальний огляд архітектури

Проект Store Olivka побудований на сучасному стеку технологій з акцентом на продуктивність та простоту підтримки.

## Технологічний стек
- **Frontend**: Next.js 15 (App Router)
- **Стилізація**: Tailwind CSS 4.0
- **База даних**: Supabase (PostgreSQL)
- **Стан**: React Context API (`CartContext.js`, `AuthContext.js`)
- **Хостинг**: Vercel

## Структура проекту
- `src/app/` — Маршрути та основні сторінки (Catalog, Product, Checkout, Admin).
- `src/components/` — Перевикористовувані компоненти та селектори доставки.
- `.agents/` — Налаштування для ШІ-агентів (правила, скіли, автоматизована пам'ять).

## Робочий процес даних
Данні завантажуються безпосередньо з Supabase. При оформленні замовлення данні записуються в БД та відправляються на вебхук n8n для автоматизації.

# Дизайн-система Store Olivka

Проект Store Olivka заснований на принципах мінімалізму, естетики та чистоти. Візуальний стиль відповідає ніші преміального одягу для немовлят.

## Кольори (Brand Colors)

| Призначення | Код | Назва |
| :--- | :--- | :--- |
| **Основний текст** | `#524f25` | Глибокий оливковий |
| **Фон сторінок** | `#faf9f6` | Теплий кремовий |
| **Акцент (кнопки)** | `#627b58` | Оливковий середній |
| **Світлий фон** | `#edf3e9` | Молочно-оливковий |
| **Акцент (золото)** | `#c4a882` | М'яке золото |

## Типографіка

В проекті впроваджено уніфіковану адаптивну типографіку:
- **Майстер-шрифт (Десктоп)**: `Century Gothic`. Використовується для **всіх** елементів: заголовків (H1-H6) та основного тексту. Це створює чистий, сучасний та преміальний вигляд.
- **Адаптивний шрифт (Мобільні)**: `Inter`. Автоматично активується через медіа-запити для екранів шириною менше за 768px для кращої читабельності.
- **Технічна реалізація**: Керується через CSS-змінні `--font-sans` та `--font-serif` у `index.css`.

## Компоненти та UI

### Картки товарів
- М'яке заокруглення кутів (`border-radius`).
- Ефект наведення (hover) на зображення.
- Лаконічне відображення ціни та статусу "Новинка".

### Кнопки
- Клас `.btn-primary`: Оливковий фон, білий текст, Century Gothic.
- Клас `.btn-outline`: Оливкова рамка, прозорий фон.
- Плавні транзиції (`transition: all 0.2s`).

# Дизайн-система Olivka Store

Усі нові компоненти мають суворо дотримуватися цих правил, щоб зберігати преміальний та ніжний вигляд сайту.

## Кольори (Brand Colors)
- **Основний текст**: `#524f25` (глибокий оливковий) — використовувати для заголовків та основного контенту.
- **Фон сторінок**: `#faf9f6` (теплий кремовий) — для `body`.
- **Акценти (Olive Palette)**:
    - `--color-olive-600`: `#627b58` (hover ефекти, кнопки).
    - `--color-olive-100`: `#edf3e9` (легкі фони блоків).

## Типографіка
- **Основний шрифт (Sans)**: `'Century Gothic', system-ui` — для звичайного тексту.
- **Заголовки (Serif)**: `'Century Gothic', Georgia, serif` — для H1-H6.
- **Особливість**: Використання 'Century Gothic' всюди для ефекту "чистоти" та елегантності.

## Принципи UI
- **Мінімалізм**: Багато вільного простору ("повітря").
- **Заокруглення**: М'які кути (border-radius) у кнопках та картках.
- **Анімації**: Плавні переходи (0.2s) та легкі мікро-взаємодії.
- **Мобільне меню**: Використовує `.no-scroll` на body для чистого скролу тільки по меню.

# 🛠️ Технічні Правила Розробки

Ці правила допомагають нам тримати проект Olivka Store стабільним і легко масштабованим.

## ⚙️ Мова та Модулі
- **Тільки JavaScript (.jsx/.js)**. Ми відмовилися від TypeScript через конфлікти з існуючою структурою.
- **Жодних `tsconfig.json`**. Якщо він з'явився — видалити негайно.
- **Шляхи Supabase**: Завжди використовувати відносний шлях: `import { supabase } from '../lib/supabase';`.

## 📦 Складання (Build)
- **Vercel Production**: Завжди деплоїти через `npx vercel --prod --yes` після важливих змін.
- **Очищення**: Якщо білд падає через старий код, перейменовувати старі папки на `_backup_..._`.

## 🔍 SEO та Метадані
- **Title**: 50-60 символів, включати брендове ім'я "Olivka Store".
- **Description**: 110-160 символів, обов'язково українською мовою.
- **OG Image**: Лише JPG/PNG 1200x630. Зберігати у `public/og-image.jpg`.
